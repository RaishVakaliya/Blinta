# Blinta (spotlightapp)

Blinta is a modern social media mobile application built with React Native and Convex. It allows users to share their thoughts, interact with others, and stay connected.

## Features

- **Authentication:** Secure user login and registration.
- **Post Creation:** Create and share posts with text and images.
- **Feed:** A chronological feed of posts from other users.
- **Bookmarking:** Save posts to a personal collection.
- **Comments:** Engage in discussions by commenting on posts.
- **Notifications:** Receive alerts for new comments and other activities.
- **User Profiles:** View and manage user profiles with profile pictures and bios.

## Tech Stack

- **Frontend:**
  - React Native
  - Expo
  - TypeScript
  - Expo Router for navigation
  - Clerk for user management

- **Backend:**
  - Convex
